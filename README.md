# Chunksky Task Breakdown Tool

## Overview
Chunksky is a task breakdown tool designed specifically for neurodivergent individuals. It assists users in breaking down complex tasks into manageable subtasks, making it easier to approach projects without feeling overwhelmed.

## Features
- **User-Friendly Interface**: Designed with accessibility in mind to ensure ease of use for everyone.
- **Customizable Task Categories**: Allows users to categorize tasks according to their preferences and needs.
- **Visual Progress Tracking**: Offers visual indicators of progress to help users stay motivated.
- **Reminders and Notifications**: Users can set reminders for each subtask to keep them on track.

## How to Use Chunksky
1. **Create a New Task**: Start by adding a primary task that you want to accomplish.
2. **Break it Down**: Use the tool to break the main task down into smaller, more manageable subtasks.
3. **Set Deadlines**: Assign deadlines to each subtask to create a timeline for your project.
